# 自研工具

Amagari Studio 不只處理翻譯文字，也會圍繞 Minecraft 在地化流程開發專用工具，把語言內容的接入、同步、校對和驗證連接成可持續維護的工作流程。目前公開介紹的第一個專案是 **Amagari Translation Tool**。

## Amagari Translation Tool

> 一個面向 Minecraft 地圖與模組在地化的 Fabric 翻譯輔助模組。它讓語言內容可以直接進入遊戲執行環境，並為多人協作、遠端同步和雙語校對提供統一支援。

**GitHub：** <https://github.com/zuiter/Amagari-Translation-Tool>

### 執行階段語言載入

工具可以讀取與 Minecraft 資源包語言檔案相容的 JSON，並將翻譯內容動態載入客戶端翻譯表。語言資料可以來自本機地圖、遠端伺服器、區域網路主機或 ParaTranz 專案，因此地圖在地化不必反覆製作和切換額外資源包。

它同時支援將同一語言拆分為多個檔案，並按照明確的優先順序組合工作階段翻譯、地圖或遠端語言、已啟用資源包和原版語言表。這樣既能保留原作資源，也能讓地圖專屬文字獲得穩定、可預期的覆蓋結果。

### 遠端與區域網路同步

在多人伺服器或區域網路地圖中，主機可以向加入者發布語言清單。客戶端只取得本機缺少或內容雜湊發生變化的資料，並透過壓縮傳輸與本機快取減少重複下載。

這套同步機制讓翻譯內容能夠跟隨伺服器版本更新，也讓參與測試的玩家保持一致的語言環境。快取按伺服器與語言隔離，並具備過期清理和歷史版本數量控制，兼顧更新效率與本機空間占用。

### ParaTranz 協作連接

工具可以連接 ParaTranz 專案，把協作平台中的語言 JSON 拉取到目前遊戲工作階段並立即套用，也可以在需要時寫入目前本機地圖語言檔案。

這使譯者能夠從線上協作、專案匯出一路走到遊戲內驗證，減少手動下載、重新命名、複製和重新打包造成的流程中斷。專案存取、匯出結果與本機快取都由工具統一管理，適合持續迭代的地圖中文化任務。

### 雙語校對與原文對照

工具提供目標語言與原始語言切換，並能在遊戲內顯示可追溯的原文。目前涵蓋的校對場景包括物品提示、帶有翻譯條目的告示牌以及書本文字。

原文顯示會盡量保留原元件的顏色、斜體、參數和排版資訊，使譯者可以在真實遊戲語境中比較譯文與原文，而不是只在表格中孤立地檢查字串。對於大廳公告、任務提示和規則說明等固定告示牌內容，工具也能透過專用翻譯條目完成客戶端顯示層的在地化，不修改地圖存檔本身。

### 工程可靠性

- 使用語言清單、內容雜湊和壓縮資料完成按需同步。
- 區分本機地圖、伺服器和不同語言的快取，避免資料互相污染。
- 自動清理長期未使用的遠端快取，並限制同一伺服器語言版本的保留數量。
- 僅向命令執行者返回私有回饋，避免多人伺服器中的操作資訊干擾其他玩家。
- 對單次遠端語言資料設定容量邊界，鼓勵地圖專案保留必要的語言和命名空間。
- 保持語言覆蓋順序明確，方便定位資源包、地圖語言與線上協作結果之間的衝突。
