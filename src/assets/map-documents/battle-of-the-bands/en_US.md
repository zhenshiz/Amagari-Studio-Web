## About the Map

Battle of the Bands is a Minecraft 1.21.11 map created by FingerMaps. Players fight enemies with musical instruments in a band showdown with dungeon-combat influences.

## Installation

1. Download the original map from [Minecraft Maps](https://www.minecraftmaps.com/51795-battle-of-the-bands) or [Planet Minecraft](https://www.planetminecraft.com/project/battle-of-the-bands/).
2. Download the localization resource pack from [Lanzou Cloud](https://wwawx.lanzouw.com/iaF9m3m3q1sj) using the password `xaleem`.
3. Place the original map folder in `saves` and the localization pack in `resourcepacks`.
4. Enable the localization pack before entering the map and place it above other packs that modify the same text.
